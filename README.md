
# FPGA Elevator Controller

This project implements a digital elevator control system using VHDL. The system simulates the operation of a multi-floor elevator and manages movement between floors based on user input.

## Features
- Finite State Machine (FSM) based control logic
- Floor selection handling
- Elevator movement control (up/down)
- Simulation of elevator states
- Implementation of request priority

## Technologies Used
- VHDL - Vivado 2025.1
- FPGA development tools - Zybo Z7 board
